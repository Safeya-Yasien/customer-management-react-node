# حيرة

## مشاكل ال edit

- ازاى هعمل ال edit هل هعمل ليها صفحه منفصلة لما ي nav عليها لما يضغط على الزرار ولا اعمل form واحدة للإضافة والتعديل وأعمل بس condition بناء على ال mode اظهر ال component
- برضوا في نفس الموضوع هى نقطة ال route name يعنى اسميه customers/edit ولا customers/:id دى برضوا أحد المشاكل الل واجهتنى

## مشاكل تسمية الملفات

- يعنى دلوقت اسمى ايه الصفحه الل فيها add customer form وهل اخلى ال form في الصفحه ولا اعملها في component منفصل

## تنظيم ال routes

- أحد الحلول الل لقيتها وفعلا جميلة هى تنظيم ال routes حيث نعمل لكل module ال routes الخاصه بيه في ملف منفصل في routes folder وبعدين ندمج كله في ملف ال AppRouter الاصلى

## تنظيم ال functions

- انى اعمل services folder الل هيحتوى على جميع customer functions فكرة كويسة جدا لل api calls
